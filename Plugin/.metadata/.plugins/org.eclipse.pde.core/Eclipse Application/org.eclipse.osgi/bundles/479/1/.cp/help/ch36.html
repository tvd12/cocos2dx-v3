Adding Screens

To add a screen to a storyboard you can use several methods:

  • Use the Add Screen… button in the editor. You can add multiple screens at once or choose to create a new one.
  • Drag & Drop screen files directly from the Project Explorer view into the editor. You may have to uncheck the Link with Editor option () in Project Explorer view first.
  • Copy & Paste screen files from the Project Explorer view into the editor.
